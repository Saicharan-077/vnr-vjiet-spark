## Getting Started

1. **First Time Setup:**
   - Go to `/login` page
   - Click "Don't have an account? Sign Up"
   - Enter your email and password
   - Account will be created with appropriate role

2. **Admin Features:**
   - After logging in as admin, you'll see "Manage Events" button in navigation
   - Click to access admin dashboard
   - Add events with complete details (title, category, date, venue, organizing club, etc.)
   - Edit or delete existing events

3. **Key Features:**
   - Modern sleek design (Steve Jobs/Apple inspired premium aesthetics)
   - Secure backend authentication with Lovable Cloud
   - Role-based access control
   - Real-time event management
   - Professional AI-generated event images
   - **Certificate Generation** - Download personalized participation certificates for any event
   - **Club Integration** - Events are associated with official college clubs
