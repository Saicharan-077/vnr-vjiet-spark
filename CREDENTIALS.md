## Getting Started

1. **First Time Setup:**
   - Go to `/login` page
   - Click "Don't have an account? Sign Up"
   - Enter your email and password
   - Account will be created with appropriate role

2. **Admin Features:**
   - After logging in as admin, you'll see "Manage Events" button in navigation
   - Click to access admin dashboard
   - Add events with complete details (title, category, date, venue, etc.)
   - Edit or delete existing events

3. **Features:**
   - Modern sleek design (SpaceX/Tesla inspired)
   - Secure backend authentication with Lovable Cloud
   - Role-based access control
   - Real-time event management
